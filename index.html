En pago parcial no se genera imagen de boleto oficial. El mensaje de WhatsApp incluira el pago realizado y el saldo pendiente.

El WhatsApp se enviara unicamente al numero del cliente capturado arriba.

WhatsApp abre el chat del cliente con el mensaje precargado. Por seguridad de WhatsApp, el usuario debe presionar enviar. La imagen del boleto oficial solo se genera cuando el estado es Ocupado total.

Tambien puedes escribir IDs separados por coma para eliminarlos de Google Sheets.
